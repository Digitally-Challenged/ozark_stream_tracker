# Ellis Creek

## Overview
**Rating:** Class III-IV
**Watershed Size:** XS
**Gradient:** Steep
**Length:** Short steep run
**Season:** Rain-dependent

## Description
Ellis Creek is a steep tributary in the Baron Fork watershed of northwest Arkansas, offering a challenging Class III-IV run when rain brings it up. Like most creeks this size, it rises fast after storms and drops quickly, so timing is everything. Technical moves, tight lines, and the usual hazards of steep creeks in the Ozarks.

## Sections

### Upper Ellis Branch
- **Character:** Steep creek
- **Features:** Class III-IV drops
- **Notes:** Rain-dependent, scout first run

## Access Points

- Located in Baron Fork watershed near Dutch Mills
- Check with local paddlers for current access

## Rapids & Features
- Steep gradient
- Technical Class III-IV drops
- Pool-drop character

## Hazards
- Wood and strainers - always scout after storms
- Rises and falls rapidly
- Remote - limited bailout options
- Intermediate to advanced paddlers

## Gauge Information
**Primary Gauge:** Baron Fork at Dutch Mills (07196900)
**Optimal Range:** 5.5-8.0 ft
**Notes:** Correlation gauge - runs when Baron Fork is high
